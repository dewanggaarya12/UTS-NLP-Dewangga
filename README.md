Sentiment Analysis (English & Indonesia) by Dewangga

## Deskripsi

Project ini melakukan analisis sentimen pada teks bahasa Inggris dan Indonesia menggunakan metode:

* Bag of Words (BoW)
* TF-IDF
* Algoritma: Multinomial Naive Bayes

## Cara Menjalankan

1. Clone repository:

```
git clone https://github.com/dewanggaarya12/UTS-NLP-Dewangga
cd UTS-NLP-Dewangga
```

2. Install dependencies:

pip install -r requirements.txt

3. Jalankan notebook:

main.ipynb

## Catatan

* Pastikan dataset tersedia di folder `data/`
* Jalankan notebook dari atas (Run All)
